# Memory Market: Truth for Sale

한국어 UI 제목: **기억시장: 진실을 사고파는 밤**

로컬 PC에서 플레이하는 React/TypeScript/Vite 기반 추리 거래 게임입니다. 현재 버전은 v1.3.1 규칙을 기준으로, 숫자 퍼즐 대신 인물, 장소, 핵심 물증, 시간대를 추리하는 구조로 동작합니다.

## v1.3.1 업데이트 요약

- 정답 구조를 숫자 1~6에서 공개 후보 카드 기반으로 변경했습니다.
- 최종 제출 항목은 `범인 / 장소 / 핵심 물증 / 시간대`입니다.
- 공식 단서는 후보의 공식 키워드와 항목 사이의 연결로 판정합니다.
- 원본 공식 기록, 공식 복사본, 증거 이용권, 원본 묶음 중복 무효 규칙을 구현했습니다.
- 라이트 사건 파일 001 `실험실의 공백 기록 - 라이트`를 고정 데이터로 포함했습니다.
- 라이트 / 스탠다드 기본 / 스탠다드 플러스 / 그랜드 / 익스트림 모드 설정을 제공합니다.

## 실행 방법

```bash
npm install
npm run dev
```

Vite가 안내하는 로컬 주소를 브라우저에서 열면 됩니다.

## 테스트 방법

```bash
npm run test
npm run build
```

테스트는 단서 판정, L001 라이트 사건 파일, 정보실 처리, 증거 이용권, 최종 채점, 모드 설정, 자동 시뮬레이션을 검증합니다.

## 모드 설명

- **라이트**: 4~6명, 4라운드, 후보 20개, 공식 기록 48장, 규칙 카드/개인 목표/자유 소문 없음, 확인 안 된 제보 사용.
- **스탠다드 기본**: 5~8명, 5라운드, 후보 24개, 공식 기록 72장, 이용 보너스 규칙 자동 적용.
- **스탠다드 플러스**: 스탠다드 기본에 간소 개인 목표, 강한 연결 주장, 제한 소문, 선택 규칙 카드를 추가.
- **그랜드**: 6~10명, 5라운드, 공식 기록 80장, 규칙 카드/개인 목표/연결 단서 5종 사용.
- **익스트림**: 8~12명, 6라운드, 공식 기록 96장, 2/4라운드 규칙 카드 코인 경쟁, v1.1 풀버전 계열.

## 라이트 사건 파일 001

사건명: **실험실의 공백 기록 - 라이트**

헬릭스 타워의 내부 조사관은 연구 자료 검토를 앞둔 밤, 실험실 인근에서 의식을 잃은 채 발견되었습니다. 조사관이 확인하던 기밀 연구 파일 일부는 사라졌고, 사건 시간대의 일부 감시 기록에는 설명하기 어려운 빈칸이 남았습니다.

라이트 정답은 앱 내부에 숨겨져 있으며, 디버그 패널에서만 확인할 수 있습니다.

## 핵심 규칙

- **정답과 증거의 분리**: 정답을 추리해도 공식 증거가 없으면 증거 점수를 얻지 못합니다.
- **원본/복사본/증거 이용권**: 원본 공식 기록은 거래 가능하고, 공식 복사본은 원본보다 기본 증거 점수가 1 낮습니다. 증거 이용권은 소유권 없이 최종 제출 때 한 번 쓸 수 있는 권리입니다.
- **확인 안 된 제보와 소문**: 공식 증거가 아니며 최종 제출에 붙일 수 없습니다. 확인 결과는 확인한 플레이어에게만 표시됩니다.
- **공개 주장**: 자유 입력이 아니라 공식 키워드 목록과 정해진 연결 문장으로만 제출합니다.
- **연결 단서**: 연결된 항목을 모두 맞혀야 증거 점수가 발생합니다.
- **원본 묶음 번호**: 같은 원본 묶음은 최종 제출 전체에서 한 번만 점수로 인정됩니다.

## 디버그 기능

- 정답 보기
- 관계 점수 보기
- 모든 공식 기록 검사
- 라이트 20회 자동 시뮬레이션
- 스탠다드 10회 자동 시뮬레이션
- 모든 모드 1회 자동 시뮬레이션
- localStorage 초기화

## 알려진 한계

- L001은 라이트 고정 데이터가 완전 구현되어 있고, 상위 모드는 같은 사건을 규칙 수량에 맞춰 생성 확장합니다.
- 그랜드 규칙 카드는 PC 진행 편의상 후보 선택 방식으로 처리되며, 익스트림은 코인 경쟁 방식입니다.
- AI는 완벽한 협상가가 아니라 게임을 끝까지 진행하는 기본 봇입니다.
- 온라인 멀티플레이와 리플레이는 아직 없습니다.
